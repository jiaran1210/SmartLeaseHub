# SmartLeaseHub - 房屋出租管理平台

## 项目背景

作为房东，需要一个高效的管理系统来统一管理多套出租房屋，包括：
- 租户基础信息管理
- 租住合同管理
- 水电煤用量与费用管理
- 房租缴纳记录与到期提醒
- 多房屋租金收入统计分析

## 功能模块

### 1. 用户认证模块

| 功能 | 描述 |
|------|------|
| 手机号注册 | 首次使用通过手机号注册账号 |
| 手机号登陆 | 通过短信验证码或密码登录系统 |
| 退出登录 | 安全退出当前账号 |

### 2. 房屋管理模块

| 功能 | 描述 |
|------|------|
| 房屋列表 | 展示所有已添加的出租房屋名称 |
| 新增房屋 | 添加新的出租房屋（地址、名称、户型等） |
| 删除房屋 | 删除房屋时需二次确认，防止误删 |
| 编辑房屋 | 修改房屋基本信息 |

### 3. 租户管理模块

| 功能 | 描述 |
|------|------|
| 租户列表 | 展示当前房屋下的所有租户 |
| 新增租户 | 添加租户信息（姓名、电话、入住时间、租期、房租等） |
| 编辑租户 | 修改租户信息 |
| 退租处理 | 记录租户退租信息 |

### 4. 合同管理模块

| 功能 | 描述 |
|------|------|
| 合同期限 | 支持按年/月设置租期 |
| 合同附件 | 可上传合同扫描件（可选） |
| 到期提醒 | 合同到期前N天提醒续租 |

### 5. 费用管理模块

#### 5.1 水电煤管理

| 功能 | 描述 |
|------|------|
| 初始值录入 | 入住时记录水电煤表底数 |
| 用量记录 | 定期记录水电煤表读数 |
| 费用计算 | 根据用量自动计算费用 |

#### 5.2 房租管理

| 功能 | 描述 |
|------|------|
| 房租缴纳记录 | 记录每次房租缴纳情况 |
| 缴纳方式 | 支持线上/线下多种缴纳方式 |
| 到期提醒 | 房租到期前提醒续缴 |

### 6. 统计分析模块

| 功能 | 描述 |
|------|------|
| 年度收入汇总 | 按年份统计各房屋租金收入 |
| 房屋收入对比 | 多房屋收入对比分析 |
| 费用趋势 | 水电煤费用月度/年度趋势 |
| 导出报表 | 支持导出Excel/PDF报表 |

## 数据模型

### 房屋 (House)
- 房屋名称/地址
- 户型（几室几厅）
- 建筑面积
- 创建时间
- 状态（出租中/空置中）

### 租户 (Tenant)
- 姓名
- 联系电话
- 身份证号（可选）
- 紧急联系人
- 入住时间
- 租期（起止日期）
- 房屋ID（关联）

### 合同 (Contract)
- 合同编号
- 租户ID（关联）
- 房屋ID（关联）
- 合同期限
- 月租金
- 押金金额
- 合同扫描件URL
- 状态（生效/已到期/已终止）

### 费用记录 (Bill)
- 类型（房租/水费/电费/燃气费）
- 关联租户
- 关联房屋
- 金额
- 记录日期
- 缴纳状态
- 缴纳日期
- 备注

### 水电煤读数 (MeterReading)
- 房屋ID（关联）
- 类型（水/电/燃气）
- 读数
- 记录日期
- 用量
- 费用

## 技术选型建议

### 前端
- **框架**: React / Vue.js
- **UI库**: Ant Design / Element Plus
- **状态管理**: Redux / Vuex / Pinia
- **图表库**: ECharts（统计分析图表）

### 后端
- **框架**: Node.js (Express/Koa) / Python (Django/Flask) / Go
- **数据库**: PostgreSQL / MySQL
- **ORM**: Prisma / Sequelize / SQLAlchemy

### 部署
- **容器化**: Docker
- **云服务**: 阿里云 / 腾讯云

## 非功能性需求

| 需求 | 说明 |
|------|------|
| 响应式设计 | PC端和移动端均可正常使用 |
| 数据安全 | 敏感信息加密存储，定期备份 |
| 性能要求 | 页面加载时间 < 3秒 |
| 兼容性 | 支持主流浏览器（Chrome、Safari、Edge） |
